# DoniStroi — Сайт фасадной компании

## Структура проекта
```
doni-stroi/
├── index.html          # Точка входа (SPA)
├── css/
│   └── style.css       # Все стили (дизайн-система + компоненты)
├── js/
│   └── app.js          # Вся логика (роутер, данные, рендер страниц)
└── README.md
```

## Страницы
| Маршрут | Страница |
|---------|----------|
| `#home` | Главная |
| `#about` | О компании |
| `#services` | Услуги |
| `#catalog` | Каталог |
| `#product/{id}` | Страница товара |
| `#contacts` | Контакты |
| `#admin` | Администрирование |

## Вход в админ-панель
- **Логин:** `admin`
- **Пароль:** `donistroi2024`

Функции: добавление, редактирование, удаление товаров (данные сохраняются в localStorage).

## Технологии
- Чистый HTML + CSS + Vanilla JS
- SPA с hash-роутингом
- localStorage для хранения каталога
- Google Fonts: Cormorant Garamond + Jost
- CSS-анимации и IntersectionObserver

## Запуск
Просто откройте `index.html` в браузере. Локальный сервер не требуется.
